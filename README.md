# Score Game Console

C# ile geliştirilmiş console tabanlı skor toplama oyunu.

## Özellikler
- Oyuncu `#` karakteri ile temsil edilir
- Ok tuşlarıyla hareket eder
- Yukarıdan düşen `*` ve `O` nesneleri yakalanır
- Skor artar
- Süre dolunca veya hedef skora ulaşınca oyun biter
- Oyun sırasında log dosyası oluşturulur

## Loglanan Olaylar
- Tuş basımı
- Oyuncu hareketi
- Obje oluşturma
- Obje hareketi
- Çarpışma kontrolü
- Skor artışı
- Oyun bitişi

## Çalıştırma
```bash
dotnet run
